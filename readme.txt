1. Run following to compile:
    1. gcc -o s server.c -lpthread
    2. gcc -o c client.c -lpthread

2. Open two terminals
3. Run ./s first and enter username
4. Run ./c in other and enter username
5. To send a message first enter 1 and then write message. End message with "%" character.
6. To end send "Good Bye!%"
